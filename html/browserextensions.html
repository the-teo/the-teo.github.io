Dark Reader

Compatible with:

Chrome Firefox Safari Edge
✔ ✔ ✔ ✔

Dark Reader is an extensions that allows you to customize the layout of any web page. If you are a fan of dark mode this is the perfect extensions for you. Dark Reader will reduce the stress of your eyes while surfing the web.

At the same time, it is easy to use and allows you to control the brightness, contrast, sepia and grayscale.

Todoist

Compatible with:

Chrome Firefox Safari Edge
✔ ✔ ✔ ✔

Todoist is an extension that allows you to create a TO DO LIST.

Since this is not a full review for the service, I will just mention that this is available, multi platform, as an app on Windows, Linux, MacOS, Apple Watch, Wear OS, android, IOS and has a dedicated extension for Email apps (Gmail, Outlook)

Todoist let's you create projects with different tasks, create a little karma chart for your accomplishments and supports repeating task periodically (such as every Friday).

It has different themes (some paid some free).
