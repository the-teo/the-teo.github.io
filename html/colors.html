try experimenting with colours yourself:

  1. RGB code

    2. RGB stands for Red, Green and Blue.

    Form: rgb(xxx,xxx,xxx)

    Where 'xxx' can have values between 0-255. 0 is the minimum, 255 is the maximum.

    Exemples of rgb code:

    pure red: rgb(255,0,0)

    white: rgb(255,255,255)

    black: rgb(0,0,0)

    purple: rgb(136,8,255)

  2. HEX code

    Form: #xxxxxx

    Where xx can be a combination of letters from A-F and numbers 0-9. Minimum is 0, maximum is F.

    First group of two #xxxxxx represents red. The second group of two #xxxxxx represents green and the last group of two #xxxxxx represents blue.

    Examples of hex code:

    pure red: #FF0000

    white: #FFFFFF

    black: #000000

    purple: #8808bb

  3. HSL code

    HSL stands for hue, saturation and lightness.

    Form: hsl(xxx, xxx%, xxx%);

    Hue represents the colour wheel, starting from 0, and ending at 360. For example 0 is red, 120 is green, 240 is blue.

    Saturation represents the intensity of the colour, represented as a percentage, starting from 0%,and ending at 100%.

    Lightness represents the white quantity in a color, represented as a percentage, starting from 0%(completely dark), and ending at 100%(completely light).

    Examples of hsl code:

    pure red: hsl(0, 100%, 50%)

    white: hsl(0, 0%, 100%)

    black: hsl(0, 0%, 0%)

    purple: hsl(283, 92%, 38%)
