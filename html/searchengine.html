What is a search engine?

A search engine is, by the definition, a program that allows you to search on the internet using key words. This program makes browsing the web easier.

Search engine scan the internet and, based on the word/words you input, show you websites that contain that information or similar one.


Exemples:

Google Search

Google search is the most known and used search engine. This one is easy to use and intuitive.

Most of the time I use Google for shopping because it has the "shop" section.

Bing

Bing is the search engine developed by Microsoft. It is similar to any other search engine.

I use Bing sometimes and I find it very similar to Google.

DuckDuckGo

DuckDuckGo is a search engine that is now popular for its promise of privacy. This search engine says that it doesn/t sell your data and blocks different trackers.

I started using DuckDuckGo as my main on my phone because it seemed more simpler to me. I find it easier to only have bookmarks, without search history or recommended search based on what you visited before.

(yes, I do have search history paused on Google too, but I still get adds about what I search even if I stopped that function)

Ecosia

Ecosia is a search engine that plants tree.

For me this one the way to go whenever I need to search different information without the need of privacy.


I know that a lot more search engines, but I wanted to include the one that I personally used and can recommend.
