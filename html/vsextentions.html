  1. Auto Rename Tag

    This specific extension is useful for coding in HTML/XML. When it is activated it will automatically rename the closing tag after the opening tag.

    Personally, since I use HTML pretty often, I find this pretty helpful and it did save a lot of time for me, so I'll highly recommend it.

    This is the extension of the theme that I use right now and have used for quite some time.

    I like it since it is entirely dark, being easier on my eyes after hours of staying in front of the computer.

  5. Prettier

    Prettier is awesome because it arranges your code when you hit save, making it easier to read and, well, prittier.

    At the same time, it makes your code look much cleaner.
