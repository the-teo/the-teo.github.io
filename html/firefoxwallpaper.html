Firefox Home Page Background

date:22/07/2021

Jump to content:

1.Firefox directory 2.Changes in the directory 3.Create files 4.Change firefox settings

I will guide you step by step, showing you the method I used to change my firefox home page background image.

1.Firefox directory

To find the firefox directory go type in your browser this:about:support

Then find:Profile Directory

and press OPEN DIRECTORY


2.Changes in the directory

Good, now we have the firefox directory.

Now, create a new directory named (remember, all letters need to be lowercase)

chrome


3.Create files

Ok, you did good until now. We are almost there.

Create a directory named

img

create a css file and save it as:

userContent.css

The content of the CSS file:

@-moz-document url(about:home), url(about:newtab), url(about:privatebrowsing) { .click-target-container *, .top-sites-list * { color: #fff !important ; text-shadow: 2px 2px 2px #222 !important ; } body::before { content: "" ; z-index: -1 ; position: fixed ; top: 0 ; left: 0 ; background: #000 no-repeat url(img/roshrach4.jpeg) center ; background-size: cover ; width: 100vw ; height: 100vh ; } }

Where you see url(img/roshrach4.jpeg) change the url to your photo location.


4.Change firefox settings

Ok, this is the last step.

Go to about:config and accept the risks

now in the search bar put toolkit.legacyUserProfileCustomizations.stylesheets and set the value to TRUE
